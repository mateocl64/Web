Mensajes de Contacto

Integración con FormSubmit

Los mensajes se envían directamente a msas@movexa.com usando FormSubmit. Esta página está preparada para futuras integraciones con bases de datos para almacenar mensajes.

Los mensajes recibidos llegarán al correo electrónico configurado en el formulario de contacto del sitio web.

Buzón de Mensajes

Actualmente los mensajes se envían directamente por email.

Para implementar un sistema de mensajes en el panel, puedes:

  • Configurar un webhook en FormSubmit para enviar mensajes a tu API
  • Crear un endpoint POST /api/messages para recibir mensajes
  • Almacenar mensajes en backend/data/messages.json
  • Mostrar los mensajes en esta página
